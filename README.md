# Smart User Platform

用户中心系统是一个高效、安全的用户管理平台，提供统一的账户管理、权限控制和身份认证功能，支持多终端、多应用的无缝对接。系统致力于提升用户体验，确保数据安全。
**核心亮点**：支持用户登录、用户注册、用户管理，基于SpringBoot/react开发。

### 快速开始
```
# 克隆仓库
git clone https://github.com/223Jiang/user-center.git

# 安装依赖
tyarn

# 运行项目
npm run dev
```
